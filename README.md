# gallery
## Description
This DApp is built with [ERC1155](https://github.com/enjin/erc-1155), which is still on development.

In this DApp, you can manage your NF tokens and buy another token.
The tokens are pictures. You can see pictures as if you were in a gallery.

In addition, you can buy pictures by "art" that is original currency in this DApp.



## Usage

![How to use the DApp](https://user-images.githubusercontent.com/38364091/54568022-08e8b780-4993-11e9-8778-b8cadca98c29.png)

1. Click what you want to bet on the table.
2. Bet ether. It must be from 0.01 ether to 0.03 ether.
3. Press the start button. When the transaction is confirmed, the roulette will run.
* Use Ropsten acccount for this dapp.
